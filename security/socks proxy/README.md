# SOCKS Proxy

## Dynamic port forwarding
```sh
ssh -D 1080 -N -p <SSH_PORT> <USER>@<REMOTE_SERVER>
```

## Browser SOCKS proxy
Use firefox > menu > settings > Network Settings > Connection Settings > Manual Proxy
```
SOCKS Host: 127.0.0.1
Port: 1080
Proxy DNS: Yes
```
